Как установить игры

  1. Нажмите на кнопку "Скачать" под выбранной игрой
  2. Дождитесь завершения загрузки архива с игрой
  3. Распакуйте архив с помощью программы WinRAR или 7-Zip
  4. Запустите файл setup.exe из распакованной папки
  5. Следуйте инструкциям установщика
  6. После завершения установки запустите игру с помощью созданного ярлыка
